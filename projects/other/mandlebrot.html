Mandlebrot

  • Description

    This is a program that plots the mandlebrot set and allows you to zoom in on any part of it. I used C++ to make this and OpenGL for the graphics.

  • Improvements

    The zooming is not working how I would like. Currently it does not zoom in on the correct spot and it moves around a ton. Also the zooming is slow since we need to recalculate the mandlebrot set on the smaller view size. I believe I can get a really good speed up if I move the calculation to the graphics card which should not be difficult if I just use a GLSL shader.
