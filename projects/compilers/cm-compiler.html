C- (C Minus) Compiler

  • Description

    This compiler is not yet on Github. I made this compiler for my Compiler Design class in Spring 2019. We were told we could use any language we wanted for this project and we could even use Flex and Bison to generate the lexer and parser. To challenge myself, I chose to use C and to implement the lexer and parser by hand. One of the problems came from when I was turning the grammar we got for the language into an LL(1) grammar. I didn't read the grammar close enough and missed something so I had to kind of force in one rule. However, my bad fix seems to work so I might not go in and redo the grammar. The other problem is with code generation. I messed up with the symbol table when generating the assembly code. This causes local variables to not work at all; so, this is a crucial fix I need to make. Once this last problem is fixed the code for the compiler will be uploaded to github and this page will be updated
