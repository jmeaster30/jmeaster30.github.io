Brainf*ck Compiler and Interpreter

  • Description

    This program is a compiler and interpreter for the esoteric programming language Brainf*ck. I chose to stick to the original specs as much as possible. So, each cell is one byte in size but I didn't like the memory size restriction so I chose to dynamically allocate memory as needed so the programmer essentially has infinite memory. I wrote this twice (once in Java and once in C) just for the fun of it really. Currently, you can translate into C or compile into MIPS assembly in the future I hope to add more targets and actually output a runnable executable.

  • Improvements

    I can probably find more optimizations to implement which would make the output executable much faster. I can also use different code generation strategies to increase executable speeds. Currently, I am using a basic stack machine to do code generation so I can probably make it so I can load more than one cell from memory at a time to reduce the number of memory reads and writes. Adding more target platforms and actually producing a runnable executable would also be good improvements for this compiler.
