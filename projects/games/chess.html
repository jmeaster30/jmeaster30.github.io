Chess

  • Description

    I made a cli chess game! My goal was to learn about Chess AI and AI / Machine Learning in general so you will be able to face off against AI with this program. I wanted to start off with a basic chess game where you can move the pieces and basically play a simple game of chess and that is what I have right now! Currently, there is no check for win conditions and no check for any of the fancier chess moves like castling and en passant. There is a command that'll give you all the possible moves a piece can make and this feature will assist with creating AI that can play chess.

  • Usage

    • Building the program

      This chess game only uses the standard c libraries so you just need to run 'make'

    • Running the program
      • ./chess You will be prompted for player names
      • ./chess [Player 1 Name] [Player 2 Name]
    • Commands
      • move [src] [dest] moves the piece from [src] to [dest]. The piece must be one of your pieces and must be able to move to the [dest] space
      • help [loc] provides a list of possible moves the piece at [loc] can move to.
      • quit quits the game

  • Improvements and Future Additions

    Currently, the commands you enter are just something I came up with but I learned about Algebraic Notation for chess so I'm thinking about converting the commands to that. The algebraic notation will also help with a feature I want to add where users can save, load, and playback games. Since, this chess notation is pretty standard in chess programs the user could even load games from this program into another program or vice versa.
