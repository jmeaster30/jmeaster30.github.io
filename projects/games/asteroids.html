Asteroids

  • Description

    This is the classic arcade game Asteroids written in C++. I used the SDL graphics library for rendering to the screen and key inputs.

  • Improvements

    I can easily implement a high score system and probably even a leaderboards system. Another improvement could be drawing optimizations which could (in the extreme case) mean migrating over to OpenGL.
