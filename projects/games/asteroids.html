Asteroids

  • Description

    This is the classic arcade game Asteroids written in C++. I used the SDL graphics library for rendering to the screen and key inputs.

  • Usage

    • Controls
      • Left Arrow: Turn Left
      • Right Arrow: Turn Right
      • Up Arrow: Accelerate Forward
      • Space: Shoot
    • How to play Asteroids in 5 easy steps

      *The given commands have only been tested on Ubuntu but the general steps should be the same for any OS

      1. Install SDL2: sudo apt install libsdl2-2.0-0 libsdl2-dev libsdl2-image-2.0-0 libsdl2-image-dev
      2. Clone Git Repo: git clone https://github.com/jmeaster30/asteroids.git
      3. Change Directory: cd asteroids
      4. Compile the C++: make
      5. Start the Game: ./asteroids
