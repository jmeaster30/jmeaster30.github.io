Demo I made this demo because I find cellular automata really fascinating and found out about continuous versions of cellular automata from this video. So, this demo generates a random set of rules, spawns some particles, and lets the simulation run. I may implement the functionality to edit the rules and allow you to place particles yourself in the future.
