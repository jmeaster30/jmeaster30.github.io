  • Cryptography

    My Mathematical Cryptography class got me interested in cryptography. Elliptic curve cryptography was awesome because elliptic curve addition is so unlike regular addition and I found it interesting that using elliptic curves over finite fields allowed for equivalent security to regular RSA encryption with smaller key sizes. We didn't learn about quantum algorithms but we discussed Shor's Algorithm and how quantum computers will break the cryptosystems we currently use, so I have been interested in quantum-proof cryptosystems and learning as much as I can about quantum computers and quantum algorithms in my free time.

  • P versus NP

    My Intro to Computation Theory class (and my Mathematical Cryptography class since we covered Integer Factorization Algorithms and Disrete Log Algorithms) got me interested in the P versus NP problem. I had known about the problem back in high school from hearing about the Millennium Prize Problems and this problem is partly my motivation for studying both math and computer science. It would be amazing if I (or anyone really) could solve this problem one day. Also, My Intro to Computation Theory professor told my class that none of us (or he said no one in general) is going to solve P versus NP and I don't like when someone says I can't do something without reasonable proof that I can't do it so I want to solve P versus NP and prove him wrong.

  • Collatz Conjecture

    The Collatz Conjecture is also an interesting problem to me. I learned about this problem thorugh my personal research of math topics and fell in love because it is so simple to describe yet it has not been proven and mathematicians say we haven't even developed the math necessary to prove this problem yet.

  • Computer Algebra Systems

    Since I have learned how to program, I liked making programs that would automate tasks that I know I could do but hated doing because they took too long and Computer Algebra Systems helped me a lot in my math and science classes when I got lazy. My abstract algebra classes were the only times where we had to use a computer algebra system because the problem would have been too hard and time-consuming to do by hand. Since I have started using them, I have been very interested in the way they work and the algorithms they use. I still don't know much about how they work but I have been trying to learn as much as I can by my own research. I have been experimenting mostly with Mathematica but I have been looking into the Sage computer algebra system.

  • Compilers, Interpreters, and Programming Language Design

    Since I started learning how to program I have been interested in how compilers and interpreters work and how programming languages are made. My class on Compiler Design gave me practice in how to go about designing a grammar for a language and implementing all of the parts to a compiler. For that class we were allowed to use tools that automated the process for us but I decided to implement all the algorithms by hand and I learned a lot about how compilers work from that. I also have written a couple interpreters for various esoteric programming languages including: Brainf*ck, Befunge-93, and Malbolge.
