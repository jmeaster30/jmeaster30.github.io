Spotify API Demo

This demo was inspired by the site billclintonswag.com. One of my friends sent me this site and asked if I could make it and decided to spend an hour or two learning the Spotify REST API. I used the p5.js javascript library for drawing the image since it provides really simple functions for rendering things. I didn't want to completely rip off the site so I added (and will add) other pictures of famous people holding vinyl records. The way I coded this lets me add them pretty easily, I only need to specify the image, the dimensions of the image, and the location of the transparency in the image and the new image will work seamlessly.

When you click the button below you will be randomly shown:

  • Bill Clinton
  • Barack Obama
